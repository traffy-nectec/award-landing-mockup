# Fondue Award - Landing Page Mockup

Repo นี้เป็น Mockup สำหรับหน้า Landing Page ของ "Fondue Award" ซึ่งแสดงผลข้อมูลสถิติของ Traffy Fondue ในรูปแบบ Dashboard

## ภาพรวม

หน้าเว็บนี้สร้างขึ้นเพื่อแสดงภาพรวมสถิติการแก้ไขปัญหาจากหน่วยงานรัฐทั่วประเทศผ่านระบบ Traffy Fondue ประกอบด้วย:

*   **สถิติหลัก:** จำนวนเรื่องแจ้งทั้งหมดและอัตราการแก้ไขสำเร็จ
*   **แผนที่ประเทศไทย:** แสดงพื้นที่ครอบคลุมการใช้งาน Traffy Fondue
*   **City Pulse:** Heatmap แสดงความหนาแน่นของการแจ้งเหตุในแต่ละวัน
*   **Live Feed:** แสดงกิจกรรมล่าสุดและเสียงสะท้อนจากประชาชน
